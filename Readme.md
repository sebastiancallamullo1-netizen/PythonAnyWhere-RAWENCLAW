# Ravenclaw - Sistema de Gestión de Productos

Proyecto desarrollado con **Django** para la gestión de productos, permitiendo listar, crear y modificar registros de manera sencilla.

## Estructura del Proyecto

- **Api/**: Contiene la lógica principal (views, models, forms, etc.)
- **Templates/**: Archivos HTML para la interfaz
- **static/**: Archivos CSS y recursos estáticos
- **Media/Productos/**: Imágenes de los productos
- **Project/**: Configuración principal del proyecto Django

## Funcionalidades

- Visualización de productos
- Creación de nuevos productos
- Edición de productos existentes
- Subida de imágenes
- Administración desde panel Django
